Migrasi ke static file

2024-12-30 08:37:25 Blogging

Saya telah melakukan migrasi dari pythonanywhere ke sini

Akhirnya saya dapat membeli domain saya menggunakan niagahoster, karena pythonanywhere.com membutuhkan akses premium untuk domain maka saya menggunakan github static file untuk hosting.

Apa saja kerepotan yang dialami?
